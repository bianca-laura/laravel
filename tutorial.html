Cuso de Laravel

Instalação

Para fazer a instalação do laravel é necessario possuir um php 7+ instalado no pc caso windows isntalar xamp.
precisa tbm do composer instalado, (apenas abrir o site do composer e fazer a instalação)
depois da o seguinte comando no caminho da pasta desejada.
composer create-project --prefer-dist laravel/laravel nomedapastaasercriada

com o laravel instaldo é so rodar o servidor usando o seguinte comando
php artisan serve
obs o servidor do apache tem q estar rodando no xamp!!
