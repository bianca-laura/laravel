Cuso de Laravel

Instalação

Para fazer a instalação do laravel é necessario possuir um php 7+ instalado no pc caso windows isntalar xamp.
precisa tbm do composer instalado, (apenas abrir o site do composer e fazer a instalação)
depois da o seguinte comando no caminho da pasta desejada.
composer create-project --prefer-dist laravel/laravel nomedapastaasercriada

com o laravel instaldo é so rodar o servidor usando o seguinte comando
php artisan serve
obs o servidor do apache tem q estar rodando no xamp!!

Rotas e views

Rotas seria a url do site ou seja o caminho desejado

Views é oq o usuario vai ver

As rotas ficam na pasta routes/web.php

por padrao a rota / é o index do site

as views ficam no caminho resources/views

por padrao vem uma blade por nome welcome

para criar uma nova view ou seja uma nova pagina visivel para o usuario basta criar uma nova blade e chama-la no routes

Blade

blade é a template engine do laravel, ou seja aonde deixamos nossa pagina dinamica

Controller

é possivel criar Controller usando o artisan usando o comando php artisan make:controller nomeDoController normalmente são usados nomes no singular
